# Averion Corporation — Landing & Press Kit

This repository contains the official marketing site for **Averion Corporation** — a technology corporation that designs and builds premium websites, applications, and scalable digital platforms.

## About the organization

Averion Corporation is a product and engineering studio focused on:

- Strategic discovery and product alignment
- Premium UI/UX systems and brand experiences
- Full‑stack delivery (web apps, APIs, infrastructure)
- Secure, observable, and scalable cloud deployments

## What’s inside

- **Landing page** with core sections (Hero, Capabilities, About, Process, Pricing, CTA)
- **Press Kit** at `/press-kit` with downloadable brand assets and copyable brand colors

## Tech stack

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion

## Running locally

```bash
npm install
npm run dev
```

## Routes

- `/` — main landing page
- `/press-kit` — brand assets and press kit
